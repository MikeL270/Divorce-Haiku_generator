Divorce Letter Haiku Generator

Welcome to Michael Lance's ridiculous Advanced Python Programming final! And yes, this was seriously turned in for college credit. This wacky tool leverages OpenAi's api module for the wacky snake language to generate hilarious (sometimes) haiku divorce letters.


This spicy webapp uses flask on the server side to interact with a custom haiku generator python module I wrote instead of having a social life. This little mess is all tied together with some swanky (new to me) javascript on the client side utilizing axios to handle asynchronous communication with the server.


It should be obvious this site is intended to be a light hearted joke, but if you actually serve one of these haikus with your divorce papers, more power to ya. (for legal reasons thats a joke)
